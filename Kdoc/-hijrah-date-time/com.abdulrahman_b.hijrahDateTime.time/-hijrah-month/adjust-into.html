adjustInto

open override fun adjustInto(temporal: Temporal): Temporal

Adjusts the specified temporal object to have this month-of-year.

This returns a temporal object of the same observable type as the input with the month-of-year changed to be the same as this.

The adjustment is equivalent to using Temporal.with passing ChronoField.MONTH_OF_YEAR as the field. If the specified temporal object does not use the Hijrah calendar system then a DateTimeException is thrown.

In most cases, it is clearer to reverse the calling pattern by using Temporal.with:

// these two lines are equivalent, but the second approach is recommended
temporal = thisMonth.adjustInto(temporal)
temporal = temporal.with(thisMonth)

For example, given a date in Jumada Al-Awwal, the following are output:

dateInJumadaAlAwwal.with(MUHARRAM);    // four months earlier
dateInJumadaAlAwwal.with(RABI_AL_THANI);      // one months earlier
dateInJumadaAlAwwal.with(JUMADA_AL_AWWAL);        // same date
dateInJumadaAlAwwal.with(JUMADA_AL_THANI);       // one month later
dateInJumadaAlAwwal.with(DHU_AL_HIJJAH);   // seven months later
