HijrahDateTime

@Serializable(with = HijrahDateTimeSerializer::class)
class HijrahDateTime : DateTimeTemporal<ChronoLocalDateTime<HijrahDate>, HijrahDateTime> , Comparable<HijrahDateTime> , TemporalAdjuster, Serializable

A date-time without a time-zone in the Hijrah calendar system, such as 1446-04-18T17:31:30.

HijrahDateTime is an immutable date-time object that represents a date-time, often viewed as year-month-day-hour-minute-second. Other date and time fields, such as day-of-year, day-of-week and week-of-year, can also be accessed. Time is represented to nanosecond precision. For example, the value "2nd Shawwal 1445 at 13:45.30.123456789" can be stored in a HijrahDateTime.

This class does not store or represent a time-zone. Instead, it is a description of the date, combined with the local time as seen on a wall clock. It cannot represent an instant on the time-line without additional information such as an offset or time-zone.
